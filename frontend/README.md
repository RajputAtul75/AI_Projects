# PFAI Frontend Application

This is the Flutter application for the Personal Finance AI.

## Prerequisites
- Flutter SDK (latest stable)
- Android Studio / Xcode for emulators.

## Setup
1. **Get Packages**
    ```sh
    flutter pub get
    ```
2. **Run Application**
    ```sh
    flutter run
    ```

## Architecture structure
- `lib/models/`: Dart data models representing backend entities.
- `lib/providers/`: State management.
- `lib/screens/`: UI Views.
- `lib/services/`: API integration and logic.
- `lib/theme/`: Shared styling.
